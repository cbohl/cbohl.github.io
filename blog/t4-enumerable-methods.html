Cycle: a Ruby Public Method


Learn some details on cycle!


February 3, 2015

Cycle is a public method that is useful for repeating a certain action for a Ruby object. It "cycles" through the object and repeats the action in a block either a specific amount of times or infinitely.

Most common use: a postitive integer argument and a block

cycle(n=nil) { |obj| block } → nil

Most commonly, cycle will be used to perform an action a given nuber of times on an object. For example, imagine we want to print out the contents of an array full of names three times.

Input:

names = ["Tom", "Ashley", "Alison", "Bryan"]
names.cycle(3) { |name| print "#{name} " }

Output:

Tom Ashley Alison Bryan Tom Ashley Alison Bryan Tom Ashley Alison Bryan => nil

Notice how the name shave been printed out, in order, three times. Also, notice how the return is "nil". Cycle will return "nil" so long as a block has been specified and the loop has not been interrupted.

No argument or non-positive integer argument, with a block

If cycle is not given any arguments, but it is given a block, the block will continual infinitely.

Input:

names = ["Tom", "Ashley", "Alison", "Bryan"] names.cycle { |name| print "#{name} " }

Output:

Tom Ashley Alison Bryan Tom Ashley Alison Bryan Tom Ashley Alison Bryan Tom Ashley Alison Bryan...

If cycle is any integer less than 1, the loop will not go through any iterations, but the method will return nil.

Input:

names = ["Tom", "Ashley", "Alison", "Bryan"] names.cycle(-2) { |name| print "#{name} " }

Output:

=> nil

With or without an argument, no block

cycle(n=nil) → an_enumerator

When no block is entered, cycle can have a slightly different use. Cycle allows you to create an enumerator, which you can then use to go through each item in the array, individually.

Input:

names = ["Tom", "Ashley", "Alison", "Bryan"] cycle_names = names.cycle

Output:

=> class #

Input:

cycle_names.next

Output:

=>Tom

Input:

cycle_names.next

Output:

=>Ashley

Input:

cycle_names.next

Output:

=>Alison

And there you have it, cycle: a method useful for running a block on a Ruby object a given number of times or creating an enumerator to cycle through the object piece by piece.
